# Photo-tagging-App
you are presented with a crowded illustration that contains many different objects. Your task is to find 2 particular characters, who are hidden somewhere in the illustration.

![preview of the app](image.png)
## Setup

### Install Dependencies
Install all dependencies for both frontend and backend:
```bash
npm i
```

### Run the Application
Start the frontend development server from the `frontend` directory:
```bash
cd frontend
npm run dev
```

Start the backend server with auto-reload from the `backend` directory:
```bash
cd backend
node --watch app.js
```
